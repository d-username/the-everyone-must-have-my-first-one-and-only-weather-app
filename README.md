## Project summary  
A mobile size weather app, using external API source and React for front-end. 
  
![weather app demo gif](https://github.com/d-username/the-everyone-must-have-my-first-one-and-only-weather-app/blob/main/public/weather_app_demo.gif)

## How to run it/set up locally  
- clone the project repository to your local machine
- in the project directory, run ``npm start``
- runs the app in the development mode
- open ``http://localhost:3000`` to view it in your browser  
  



## What technology I used  
- HTML
- CSS
- React
- JavaScript
- external API source: https://openweathermap.org/api  
  



## Feature list  
- the user can search for locations, cities
- the temperature unit can be toggled between Celsius and Fahrenheit
- the image and theme colour change based on the current weather and daytime
- the greeting and day displayed are responsive to the actual day and local time at the location we searched for  
  



## Possible additional features  
- the whole app could be more responsive to different screen sizes
- additional weather info can be integrated, such as humidity, wind speed, sunset time
